# LLM Zoomcamp — Course Work & Projects

This repository contains my personal notes, code, hands-on module exercises, and homework assignments for the **[LLM Zoomcamp](https://github.com/DataTalksClub/llm-zoomcamp)**, a free 10-week intensive course organized by [DataTalks.Club](https://datatalks.club/).

The primary goal of this course is to learn how to build practical, production-ready applications utilizing Large Language Models (LLMs), with a strong focus on Retrieval-Augmented Generation (RAG), vector databases, search evaluation, and orchestration.

## 📂 Repository Structure

The repository is organized by course modules. For each module, there is a course folder and a homework folder containing class and homework files respectively.

*   📂 **`module1_agentic-rag/`** - Agentic RAG, basic keyword search, and building an initial pipeline.
*   📂 **`module1_homework/`** - Assignment on an agentic RAG for lesson notes using chunked search.
*   🚀 **`capstone-project/`** - *[Coming Soon]* My final custom end-to-end RAG application.

## 🎯 What I'm Learning & Building

Throughout this cohort, I am building hands-on experience with:
- **RAG Architecture:** Building scalable systems to retrieve relevant context and generate grounded answers.
- **Search Technologies:** Implementing keyword search, vector embeddings, hybrid search, and reranking.
- **Evaluation & Monitoring:** Using quantitative metrics and LLM-as-a-Judge paradigms to keep applications reliable.
- **Production Deployment:** Packaging pipelines using containers (Docker) and deploying clean user-facing APIs/interfaces.
